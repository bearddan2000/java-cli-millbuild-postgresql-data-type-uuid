# java-cli-millbuild-postgresql-data-type-uuid

## Description
Creates a small table `dog` that uses
a uuid data type.

A java gradle build, that connects to postgresql database.

## Tech stack
- java
- millbuild
  - log4j
  - postgresql drivers

## Docker stack
- postgresql:alpine
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
